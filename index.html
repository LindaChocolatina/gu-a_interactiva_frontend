Sobre mí

Este es mi segundo ejercicio creando una página web usando HTML5. Mi primera web es un blog llamado "Sobreviviendo a las vacaciones" que hice como tarea de vacaciones que nos dejó el profesor Julio. Ahora estoy practicando etiquetas semánticas como header, main y section con la guía interactiva del profesor Miguel.

Mi proceso de aprendizaje

Estoy aprendiendo a estructurar mejor mi contenido usando etiquetas semánticas. Me gusta mucho la idea de usar sections para organizar diferentes partes de mi página, como enlaces, multimedia y formularios. También estoy experimentando con tablas para mostrar información de manera más clara.

Actualidad

Actualmente estoy enfocada en aprender HTML5 y CSS para poder crear páginas web más atractivas y funcionales. Mi objetivo es seguir practicando y eventualmente aprender JavaScript para agregar interactividad a mis proyectos. También me gustaría aprender sobre accesibilidad web para asegurarme de que mis páginas sean inclusivas para todos los usuarios y mejorar mis habilidades en diseño web para crear interfaces más atractivas y fáciles de usar, por esa razón quiero aprender a usar herramientas como Figma o Adobe XD para diseñar prototipos antes de codificar. Me interesaría trabajar en la experiencia del usuario (UX) y la interfaz de usuario (UI) para crear sitios web que no solo sean funcionales, sino también agradables de usar. Además quiero que todo mi aprendizaje me ayude a en el desarrollo de productos digitales que sean útiles y atractivos para los usuarios, y que me permitan crecer profesionalmente en el campo del desarrollo web.
